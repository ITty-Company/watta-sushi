# Суши Ватта - iOS Приложение

## Требования
- Xcode 15.0 или выше
- iOS 17.0 или выше
- macOS для разработки

## Установка

1. Откройте проект в Xcode:
```bash
cd ios/SushiVatta
open SushiVatta.xcodeproj
```

2. Выберите симулятор или подключенное устройство в Xcode

3. Нажмите Run (⌘R) или выберите Product > Run

## Структура проекта

- `SushiVattaApp.swift` - точка входа приложения
- `ContentView.swift` - основной интерфейс с вкладками и всеми экранами

## Функциональность

- Меню с категориями блюд
- Корзина для добавления товаров
- Профиль пользователя

## Примечания

Для запуска на реальном устройстве потребуется:
- Apple Developer аккаунт
- Настройка сертификатов в Xcode

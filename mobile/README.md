# WATTA SUSHI - React Native Mobile App

Полноценное мобильное приложение на React Native для доставки суши, полностью повторяющее дизайн iOS версии.

## Структура проекта

```
mobile/
├── src/
│   ├── components/
│   │   ├── MenuView.tsx      # Главный экран меню
│   │   ├── MenuItemCard.tsx  # Карточка товара
│   │   ├── CartView.tsx      # Корзина
│   │   └── ProfileView.tsx   # Профиль
│   └── context/
│       └── CartContext.tsx   # Контекст корзины
├── App.tsx                   # Главный компонент с навигацией
├── index.ts                  # Точка входа
└── package.json
```

## Установка и запуск

### 1. Установите зависимости

```bash
cd mobile
npm install
```

### 2. Для iOS

```bash
cd ios
pod install
cd ..
npm run ios
```

### 3. Для Android

```bash
npm run android
```

### 4. Запуск Metro bundler

```bash
npm start
```

## Особенности

- ✅ Полный дизайн из SwiftUI версии
- ✅ Навигация через Tab Navigator
- ✅ Управление корзиной через Context API
- ✅ Анимации и переходы
- ✅ Адаптивный дизайн
- ✅ TypeScript поддержка

## Компоненты

### MenuView
Главный экран с:
- Хедером с логотипом WATTA SUSHI
- Горизонтальной навигацией категорий (пилюли)
- Промо-баннером с супами
- Списком товаров с карточками

### MenuItemCard
Карточка товара с:
- Изображением (эмодзи)
- Названием и описанием
- Ценой
- Кнопкой добавления в корзину
- Бейджем "Топ" для популярных товаров

### CartView
Экран корзины с:
- Списком добавленных товаров
- Подсчетом общей суммы
- Кнопкой оформления заказа

### ProfileView
Экран профиля с:
- Информацией о пользователе
- Историей заказов

## Технологии

- React Native 0.76.0
- React Navigation 6
- TypeScript
- Context API для состояния

# Суши Ватта - Android Приложение

## Требования
- Android Studio Hedgehog или выше
- JDK 8 или выше
- Android SDK 24+ (минимум), 34 (целевой)

## Установка

1. Откройте папку `android` в Android Studio

2. Дождитесь синхронизации Gradle (может занять несколько минут при первом запуске)

3. Выберите эмулятор или подключенное устройство

4. Нажмите Run (▶️) или Shift+F10

## Структура проекта

- `MainActivity.kt` - главная активность приложения
- `MainScreen.kt` - основной UI с навигацией и всеми экранами
- `ui/theme/` - тема приложения (цвета, типографика)

## Функциональность

- Меню с категориями блюд
- Корзина для добавления товаров
- Профиль пользователя

## Примечания

При первом запуске Android Studio автоматически загрузит необходимые зависимости через Gradle.

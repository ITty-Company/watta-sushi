# ℹ️ Информация о предупреждениях сборки

## "Ignoring duplicate libraries: '-lc++'"

Это **не критичное предупреждение**. Оно означает, что библиотека C++ (`-lc++`) добавлена несколько раз в настройки линковки, но линкер автоматически игнорирует дубликаты.

### Можно ли игнорировать?

✅ **Да, можно игнорировать.** Это стандартное предупреждение React Native проектов и не влияет на работу приложения.

### Если хотите убрать предупреждение:

1. Откройте проект в Xcode
2. Выберите проект → Target **SushiVattaRN**
3. Вкладка **Build Settings**
4. Найдите **Other Linker Flags**
5. Удалите дубликаты `-lc++` (оставьте только один)

Но это не обязательно - предупреждение не влияет на функциональность.

## Другие предупреждения

Если видите другие предупреждения (deprecated APIs, unused code и т.д.) - это тоже нормально для React Native проектов. Они не критичны и не мешают работе приложения.

## Статус сборки

Если сборка завершается успешно (Build Succeeded) - всё в порядке! Предупреждения можно игнорировать.
